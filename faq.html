Why is DiscordSource Unsafe?

Discord Source and BetterDocs are unsafe because the owner, (0mniscient) posts plugins that can make your account vunerable by exposing the token. These include plugins such as Account Switchers, bot logins or token logins. Back about a year ago, he didn't check a plugin for malicious code and posted a token stealer on BetterDocs. We (the BetterDiscord Community) got the blame for the lost accounts. Added to this, he has other plugins that fit the "unsafe" categoy. Plugins such as message loggers, nodeletemessages and other such plugins that would interact with the API are on his site. This wouldn't get your account STOLEN but it is against the API terms, which is something discord takes seriously if broken.



What is the difference between BetterDocs, Discord Source and BetterDiscord?

BetterDocs was a website unaffiliated with BetterDiscord used to distribute themes and plugins. Discord Source is simply a rebranded BetterDocs with a different domain. BetterDiscord is a program used to modify the discord client. BetterDocs and Discord Source are deemed unsafe and misleading, which is why you are reading this FAQ.

What does the blocker do in all? Are there any screenshots?

The blocker does the following:

  • Displays a warning whenever a link to BetterDocs, Discord Source, or an invite to the Discord Source server is sent.
  • Makes links to BetterDocs and Discord Source non-clickable.
  • Blocks their theme preview system.
  • Prevents the creator of the website (0mniscient/Mr. Robot) from using your theme.

    • You can also customize the blockers look, making it fit your theme.



    Why is BetterDiscordLibrary safe but Discord Source isn't?

    Although BetterDiscordLibrary isn't associated with BetterDiscord themselfs, it is a safe source because the owner (Gibbu) only posts content that has already been featured on the BD server and always asks for permission to post. If Discord Source did that, we would have no problem with it.



    How can I help against this?

    If you have little knowledge in CSS/JS, then you can help by telling anyone you see who is using these sources about it's danger. If you do have some programming experience, you can join our server (link at the top bar), and help with the development of the blocker. If you are a theme developer, you can help by including the blocker in your themes.
